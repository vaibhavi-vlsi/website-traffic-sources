# Website Traffic Sources

This project analyzes website traffic data and visualizes the distribution of traffic sources using Python.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Load website traffic data from a CSV file
- Calculate total website visits
- Analyze traffic sources
- Visualize traffic distribution using a pie chart

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the script:

python traffic_analysis.py

## Output

The project generates a pie chart named `traffic_sources.png` showing the percentage contribution of each traffic source.
